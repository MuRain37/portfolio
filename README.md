# 刘佳鑫个人作品集

面向机器人、ROS2、计算机视觉和嵌入式 AI 岗位的静态作品集网站。

## 本地预览

在当前目录启动任意静态服务器，例如：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

## GitHub Pages 部署

1. 创建 GitHub 仓库，推荐命名为 `用户名.github.io`。
2. 将本目录内的全部文件推送到仓库根目录。
3. 在仓库 `Settings > Pages` 中选择从 `main` 分支根目录部署。
4. 等待 GitHub Pages 构建完成后访问生成的网址。
