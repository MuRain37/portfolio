PROJECT 02

项目负责人

基于 OpenCV 与 FOC 的
激光打靶小车

面向全国大学生电子设计竞赛 TI 杯 E 题，搭建基于树莓派、STM32/MSPM0G3507、 OpenCV 与 FOC 控制的视觉打靶小车。作为项目负责人，负责视觉识别、云台控制、 串口通信协议与打靶流程联调。

  • 基于动态 ROI、透视变换与靶纸轮廓检测完成方框扫描、目标定位和激光瞄准。
  • 结合卡尔曼滤波、线性运动预测、PID 调参和平滑控制，提升追踪稳定性。
  • 构建多处理器串口通信协议，将视觉计算、底盘控制与云台控制解耦。
OpenCV STM32 MSPM0 FOC PID 视觉闭环

PROJECT 03

项目负责人

SoftBank NAO
人形机器人高尔夫

面向机器人高尔夫赛道，基于 NAOqi SDK、OpenCV 与 Python 多线程开发自主寻球、 对准和击球流程。作为项目负责人，负责视觉识别策略、运动决策和比赛现场联调。

  • 使用 HSV 双段掩码、DoG 边缘增强、圆度过滤与霍夫变换，提高红球识别稳定性。
  • 基于目标跟踪、单目/双目测距和视觉矫正闭环，完成球距估计与路径修正。
  • 设计自主寻球、姿态调整与击球策略，获得湖南省机器人高尔夫赛道二等奖。
NAOqi SDK Python OpenCV 视觉测距 多线程

PROJECT 04

项目负责人

AIDoLamp 智能台灯

面向桌面陪伴式交互场景，搭建集视觉识别、手势交互与机械臂跟随于一体的智能台灯原型。 作为项目负责人，负责整体方案设计、视觉算法集成、控制逻辑设计与演示联调。

  • 基于 YOLOv8n 完成目标识别，结合 MediaPipe 实现手势触发与交互控制。
  • 使用 OpenCV 处理摄像头画面，输出目标位置并驱动灯体机械臂完成跟随。
  • 完成实物结构、感知链路和演示流程集成，形成可展示的桌面 AI 硬件原型。
YOLOv8n MediaPipe OpenCV 机械臂追踪 多模态交互
结构展示 展示智能台灯原型的整体外观、摄像头模组与机械臂结构细节
